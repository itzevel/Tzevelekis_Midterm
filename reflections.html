Development Reflections

Reflection 1: Learning the Basics

I began my journey with HTML and CSS, learning how to structure a webpage and style it effectively. This foundational knowledge has been critical in building more complex layouts.

Reflection 2: JavaScript and Interactivity

JavaScript allowed me to add interactivity to my projects. Understanding the Document Object Model (DOM) was crucial for manipulating elements on the page and enhancing user experience.

Reflection 3: Exploring Frameworks

Learning frameworks like Bootstrap and React has accelerated my development process. These tools provide pre-built components and structures that streamline project development and enhance responsiveness.

Reflection 4: Debugging and Problem-Solving

Debugging is an essential part of development. It taught me how to approach problems logically, test solutions, and understand the importance of writing clean, maintainable code.
